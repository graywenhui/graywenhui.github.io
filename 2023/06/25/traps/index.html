Backtrace (moderate)

要求我们实现一个获取回溯信息的函数。

课程网站上给的信息足够多，写出来还是比较简单的。

网站告诉我们GCC编译器将当前执行函数的帧指针存储在寄存器s0中。在kernel/riscv.h中添加以下函数： static inline uint64 r_fp() { uint64 x; asm volatile("mv %0, s0" : "=r" (x) ); return x; }，所以r_fp()就可以获得帧指针。
